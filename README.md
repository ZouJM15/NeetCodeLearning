# NeetCode 学习记录

## NeetCode 150是什么？

Neetcode 150 是一份精选的 LeetCode 题目列表，包含 150 道题目，涵盖数据结构的所有主题。对于准备应聘公司可能遇到的 LeetCode 测试的朋友来说，是一份宝贵的学习资源。题目分为多个类别，每个类别下又包含简单、中等和困难三个难度级别。

Neetcode的 [原始存储库](https://github.com/neetcode-gh/leetcode) 和 [网站](https://neetcode.io/)。
